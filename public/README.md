# Папка для изображений

Поместите файл логотипа с именем `logo.png` в эту папку.

Логотип будет автоматически отображаться в шапке сайта.

**Поддерживаемые форматы:** PNG, JPG, JPEG, SVG, WebP

**Рекомендуемый размер:** примерно 100x40 пикселей (или пропорционально)

Если вы хотите использовать другое имя файла или формат, измените путь в файле `components/Header.tsx`:

```tsx
<Image
  src="/ваше-имя-файла.png"  // измените здесь
  alt="ПЖ19"
  width={100}
  height={40}
  className={styles.logoImage}
  priority
/>
```
